1.在类和结构体外部的static

报错，多重符号，因为s_var是全局变量，会跨翻译单元编译
修改为
Static.cpp:
static int s_var = 5;
即让Static.cpp的s_var不能被main.cpp可见，也就不会链接导致多重定义的问题
所以在做一个大的项目时，尽量少用全局变量，多用静态变量
